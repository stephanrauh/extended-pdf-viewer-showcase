To open a password protected file, you can either have the user type the password or you can pass the password programmatically.

[password]="undefined" (or just omit it) (forces the user to enter the password) password="graalvm-rocks!" (correct password, the file opens) password="graalvm-sucks!" (wrong password, standard password prompt shows) password="graalvm-sucks?" (wrong password - custom password dialog shows)

You can also implement a custom password dialog and register it as a pdfDefaultOption. The dialog has to be a object implementing the interface PasswordPrompt. This demo implements a very simple custom password prompt: it simply uses the confirm() function.
