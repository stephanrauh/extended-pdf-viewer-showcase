The NgxExtendedPdfViewerService provides programmatic control over the PDF viewer. This service allows you to interact with the viewer, manipulate pages, control editors, and perform various operations on the PDF document.

Quick Start

Inject the service into your component and use it after the view has initialized:

import { Component, inject, AfterViewInit } from '@angular/core';
import { NgxExtendedPdfViewerService } from 'ngx-extended-pdf-viewer';

@Component({
  // component configuration
})
export class MyComponent implements AfterViewInit {
  private pdfService = inject(NgxExtendedPdfViewerService);

  ngAfterViewInit() {
    // Service is ready to use
  }
