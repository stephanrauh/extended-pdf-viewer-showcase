The PDF viewer supports Content Security Policy (CSP) by default. However, if you don't need CSP, the PDF viewer can load the PDF file faster if you disable CSP. To be hones, at the moment the diffence is only a single request, but if the idea carries fruit, maybe there's more to come.

You can observe the difference in the network tab of the developer tools of your browser. If "use inline scripts" is checked, the file op-chaining-support.js is not loaded from the server. If "use inline scripts" is unchecked, the file is loaded.
