The book mode displays the PDF file like a book and renders a nice page-turn effect.

Use [showPageFlipButton]="true" together with [showHandToolButton]="true" to get a three-mode cursor tool in the toolbar: text selection, hand tool (pan), and page flip. When entering book mode, the page-flip mode activates automatically.

Additional options:

  • [enableFlipByDrag]="false" disables drag-to-flip without showing the toolbar buttons, allowing text selection and panning. Click-to-flip still works.
  • [showPageCorners]="false" hides the dog-ear fold animation on page corners.
  • Drag-to-flip is automatically disabled when zoomed in beyond page-fit scale.

Book mode suffers from a couple of limitations:

  • Presentation mode doesn't work.
  • If the available space of the viewer isn't wide enough, weird effects happen. That's become better with version 23.2.0, but it's still not perfect.
