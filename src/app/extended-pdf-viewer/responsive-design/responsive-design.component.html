The PDF viewer uses responsive design to adapt to different screen sizes. On large screens, the toolbar displays many buttons. On smaller screens, some buttons move to a secondary menu. Button Visibility Control

You can control when buttons appear by setting breakpoints. For example, showXXXButton="xs" makes a button visible when the PDF viewer is at least 490px wide. The available breakpoints are xs: 490px, sm: 560px, md: 610px, lg: 660px, xl: 740px, xxl: 830px.
