You can customize the thumbnails. To do so you need to define an Angular template - or something that looks like an Angular template.

Caveat: As of version 4.0.0, the template does not participate in the Angular interpolation. You can't use the interpolation, nor can you use Angular events. This demo shows how to circumvent this restriction.

You can use these variables:

  • PAGE_NUMBER
  • WIDTH_OF_RING width of the ring around the thumbnail. This ring is used to highlight the selected page.
  • HEIGHT_OF_RING height of the ring around the thumbnail.
